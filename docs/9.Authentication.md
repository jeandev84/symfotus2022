### Authentication

1. Разница Аутентификация и Авторизация
```
==========================================================================
Аутентификация : 
==========================================================================

Это способ определение личности пользователя, 
это тот каким образом мы узнаем что за пользователь к нам пришёл. 
(Роль, какие права у него)
как может проводиться:
- Ввход логина-пароля
- Куки
- Токен (JWT, Ordinaire Token)

Хранилище пользователей
- База данных
- Конфиг
- Внешние источники
 * Внешняя сессия
 * JWT-Токен            
                 
                 
============================================================================================
Авторизация  : 
============================================================================================

Это определение прав доступа аутентифицированного пользователя
Способы авторизации :
- ACL (Access Control List)                 * список пользователей с правом доступа
- RBAC (Role Base Access Control)           * список ролей с правом доступа
- ABAC (PBAC) (Access Base Access Control)  * список правил, опрледеляющих права доступа
- RBAC + PBAC                               * права доступа на основании роли и контекста
```
